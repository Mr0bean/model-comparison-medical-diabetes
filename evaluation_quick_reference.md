# 人工评测 - 快速参考指南

## 📋 评分维度速查表

| 维度 | 满分 | 权重 | 核心问题 |
|------|------|------|----------|
| **信息完整性** | 20 | 20% | 关键信息有没有遗漏? |
| **信息准确性** | 25 | 25% | 提取的信息对不对? |
| **结构与组织** | 15 | 15% | 信息组织得好不好用? |
| **临床相关性** | 20 | 20% | 突不突出重点,助不助于决策? |
| **语言表达** | 10 | 10% | 表达专不专业、简不简洁? |
| **整体可用性** | 10 | 10% | 临床能不能直接用? |
| **总分** | **100** | **100%** | - |

## ⚡ 30秒快速评分法

### Step 1: 快速通读 (30秒)
- 阅读模型输出
- 形成整体印象
- **不要急于打分**

### Step 2: 核对信息 (3分钟)
对照原始记录,勾选清单:

**必需信息**:
- [ ] 基本信息(年龄、性别、身高、体重)
- [ ] 主诉
- [ ] 病程时间
- [ ] 起病方式
- [ ] 当前用药(药名+剂量+频次)
- [ ] 既往用药
- [ ] 血糖数值
- [ ] 近期变化(体重、住院等)
- [ ] 既往史
- [ ] 家族史

**信息准确性**:
- [ ] 数值准确(年龄、身高、体重、血糖)
- [ ] 时间准确(病程、事件时间点)
- [ ] 药物准确(药名、剂量、频次)
- [ ] 无无中生有

### Step 3: 评估结构 (1分钟)
- [ ] 有清晰分节
- [ ] 有列表或加粗
- [ ] 时间线清楚
- [ ] 一眼能找到需要的信息

### Step 4: 评估临床价值 (2分钟)
- [ ] 重点信息突出(如血糖不佳、体重下降)
- [ ] 有风险提示或临床建议
- [ ] 信息聚合好(如所有用药放一起)

### Step 5: 打分 (2分钟)
按照评分标准打分

### Step 6: 记录理由 (1分钟)
写下主要问题或亮点

**总耗时**: 约10分钟/案例

## 🎯 评分档次速查

### 信息完整性 (20分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 20 | 优秀 | 10个关键点全部覆盖 |
| 15-19 | 良好 | 缺1-2个次要信息 |
| 10-14 | 中等 | 缺3-4个信息或1个重要信息 |
| 5-9 | 较差 | 多个重要信息缺失 |
| 0-4 | 很差 | 大量关键信息缺失 |

**关键点清单** (10项):
1. 基本信息
2. 病程时间
3. 起病方式
4. 当前用药
5. 血糖数值
6. 体重变化
7. 住院史
8. 既往用药
9. 既往史
10. 家族史

---

### 信息准确性 (25分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 25 | 优秀 | 无任何错误 |
| 20-24 | 良好 | 1-2处轻微错误 |
| 15-19 | 中等 | 3-4处错误或1处严重错误 |
| 10-14 | 较差 | 多处错误 |
| 0-9 | 很差 | 大量或严重错误 |

**扣分标准**:
- 轻微错误(不精确): -1分
- 中等错误(时间/剂量误): -3分
- 严重错误(数值/药名误、无中生有): -5分

---

### 结构与组织 (15分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 15 | 优秀 | 分节清晰,列表规范,一目了然 |
| 12-14 | 良好 | 基本清晰,略有混乱 |
| 9-11 | 中等 | 分节存在但不够清晰 |
| 6-8 | 较差 | 分节混乱,堆砌 |
| 0-5 | 很差 | 完全无结构 |

**检查点**:
- [ ] 有"基本信息""主诉""现病史"等分节
- [ ] 用药用列表展示
- [ ] 时间线清楚(如:12年前→5月前→出院后→近期)
- [ ] 使用加粗、标题、列表等格式

---

### 临床相关性 (20分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 20 | 优秀 | 重点突出,有临床导向总结 |
| 16-19 | 良好 | 重要信息基本突出 |
| 12-15 | 中等 | 信息平铺,重点不够 |
| 8-11 | 较差 | 重要信息淹没 |
| 0-7 | 很差 | 完全无临床导向 |

**加分项**:
- ✓ 有"重点关注"或"需要注意"板块
- ✓ 突出警示信息(血糖不佳、体重下降、并发症线索)
- ✓ 聚合相关信息(所有用药集中)
- ✓ 有临床建议或风险提示

---

### 语言表达 (10分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 10 | 优秀 | 专业、简洁、流畅 |
| 8-9 | 良好 | 基本专业,略有冗余 |
| 6-7 | 中等 | 一定赘述或用词不当 |
| 4-5 | 较差 | 多处赘述或口语化 |
| 0-3 | 很差 | 表达混乱,非常不专业 |

**扣分点**:
- ❌ 口语化:"有点高""吃药""很多年"
- ❌ 冗余:"患者的糖尿病疾病"
- ❌ 不专业:"尿里有泡泡""血糖不好"

---

### 整体可用性 (10分)

| 得分 | 档次 | 特征 |
|------|------|------|
| 10 | 优秀 | 可直接用于临床,无需修改 |
| 8-9 | 良好 | 稍作修改即可使用 |
| 6-7 | 中等 | 需要一定修改 |
| 4-5 | 较差 | 需要大量修改 |
| 0-3 | 很差 | 基本不可用,需重写 |

**评估问题**:
1. 能否帮你快速了解患者情况?
2. 需不需要再看原始记录?
3. 能否基于此制定诊疗计划?

## ⚠️ 常见错误类型及扣分

### 完整性错误

| 错误类型 | 示例 | 扣分 |
|----------|------|------|
| 缺少病程 | 未提"12年" | -2 |
| 缺少用药 | 仅提2种药,实际4种 | -3 |
| 缺少血糖数值 | 仅说"偏高" | -2 |
| 缺少体重变化细节 | 未提"10余斤" | -1 |
| 缺少家族史成员 | 仅说"有家族史" | -2 |

### 准确性错误

| 错误类型 | 示例 | 严重度 | 扣分 |
|----------|------|--------|------|
| 病程时间错 | "10年"(实际12年) | 严重 | -5 |
| 数值错误 | 血糖"7"(实际11) | 严重 | -5 |
| 药名错误 | "格列美脲"(实际格列齐特) | 严重 | -5 |
| 剂量/频次错 | "一天一次"(实际早晚各一次) | 中等 | -3 |
| 时间点错 | "3个月前"(实际5个月前) | 中等 | -3 |
| 不精确 | "50岁"(实际50-55岁) | 轻微 | -1 |
| 单位缺失 | "血糖11"(缺mmol/L) | 轻微 | -1 |
| 无中生有 | 编造未提及的信息 | 严重 | -5 |

## 📝 评分记录技巧

### 简化记录法

**不要写**:
```
信息完整性得分18分,因为缺少了剂量不详的说明,
年龄表述为50多岁不够精确,应该写成50-55岁...
```

**建议写**:
```
完整性 18/20
- 缺: 剂量不详未标注
- 不精确: "50多岁"→"50-55岁"
```

### 快速标注符号

- ✓ = 做得好
- ✗ = 有错误/缺失
- ? = 不确定,需复核
- ! = 严重问题

**示例**:
```
✓ 结构清晰
✓ 用药完整
✗ 病程错误(10→12)
! 血糖数值错误
? 体重单位待核实
```

## 💡 评分提示

### 边界案例处理

**案例1: 信息提取为范围**
```
原文: "50多岁"
模型: "50-55岁" ✓ 可接受
模型: "约50岁" △ 轻微不精确(-1)
模型: "50岁" ✗ 错误(-3)
```

**案例2: 合理推断**
```
原文: "小便泡沫多"
模型: "小便泡沫多,疑似蛋白尿" ✓ 临床相关性+分
模型: "确诊糖尿病肾病" ✗ 无中生有(-5)
```

**案例3: 用词替换**
```
原文: "司美格鲁肽"
模型: "司美格鲁肽" ✓
模型: "GLP-1受体激动剂司美格鲁肽" ✓ 更专业
模型: "降糖针" ✗ 不准确(-3)
```

## 📊 评分统计速查

### 个人评分分布参考

**健康的评分分布** (70个案例):
- 90-100分: 10-15% (7-10个)
- 80-89分: 25-35% (18-25个)
- 70-79分: 30-40% (21-28个)
- 60-69分: 15-25% (10-18个)
- <60分: <10% (<7个)

**如果你的分布偏离较大**:
- 全都>85分 → 可能太宽松
- 全都<70分 → 可能太严格
- 没有明显差异 → 可能区分度不够

### 时间管理建议

- **单案例**: 8-12分钟
- **每小时**: 5-6个案例
- **每天**: 15-20个案例(含休息)
- **每周**: 20-25个案例(兼职评测)
- **总计**: 3-4周完成70个案例

## 🆘 疑难问题快速解答

**Q: 模型输出了原文没有的合理推断,算不算错?**
A:
- 如果是临床常识(如BMI计算、风险提示) → 临床相关性加分
- 如果是无依据的诊断/数值 → 准确性扣分(-5)

**Q: 两个模型都差不多,怎么区分?**
A:
- 看细节准确度(年龄范围 vs 模糊)
- 看结构清晰度(多级标题 vs 平铺)
- 看临床导向(有重点提示 vs 无)

**Q: 模型用了非常专业的术语,但原文是口语,算对吗?**
A:
- 专业化口语(如"小便泡沫"→"蛋白尿") ✓ 可接受,甚至加分
- 过度专业化(如"糖尿病"→"2型糖尿病伴周围神经病变") ✗ 若无依据则扣分

**Q: 遇到完全看不懂的输出怎么办?**
A:
- 标记为"需复核"
- 暂时打低分(如40-50分)
- 在总评中说明"输出混乱,难以理解"
- 提交给质控员审查

**Q: 自己前后评分标准不一致怎么办?**
A:
- 每评10个案例,休息并回顾标准
- 发现偏差后,调整后续评分
- 可选择性复评前面案例(特别是边界分数的)
