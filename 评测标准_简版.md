# 医疗AI模型输出评测标准（简版）

## 📊 评分体系总览

**总分：100分 = 5个维度**

| 维度 | 分值 | 核心要求 |
|------|------|---------|
| **信息准确性** | 30分 | 无错误、无曲解 |
| **信息完整性** | 25分 | 无关键遗漏 |
| **临床实用性** | 20分 | 突出重点、助决策 |
| **结构清晰度** | 15分 | 层次分明、易查找 |
| **语言专业性** | 10分 | 专业简洁 |

---

## 1️⃣ 信息准确性（30分）

**评分原则：从30分开始，发现错误即扣分**

### 常见错误扣分

| 错误类型 | 扣分 | 示例 |
|---------|------|------|
| 数值完全错误 | -5分 | "血糖11"写成"血糖7" |
| 药名错误 | -5分 | "二甲双胍"写成"格列美脲" |
| 无中生有 | -5分 | 编造不存在的检查结果 |
| 剂量/频次错误 | -3分 | "早晚各1次"写成"每日1次" |
| 时间点错误（>3年） | -3分 | "12年"写成"5年" |
| 数值不精确 | -1分 | "11 mmol/L"写成"约11" |

### 检查清单
- [ ] 患者基本信息（性别、年龄、身高、体重）
- [ ] 病程时间
- [ ] 药物名称、剂量、频次
- [ ] 检查数值及单位
- [ ] 无编造信息

---

## 2️⃣ 信息完整性（25分）

**评分原则：覆盖率 = 提取信息数 / 应有信息数**

### 覆盖率对应分数

| 覆盖率 | 得分 | 说明 |
|--------|------|------|
| ≥95% | 23-25分 | 仅遗漏1-2个次要信息 |
| 90-94% | 19-22分 | 遗漏2-3个次要信息 |
| 80-89% | 14-18分 | 遗漏3-4个信息 |
| 70-79% | 8-13分 | 遗漏较多或1个重要信息 |
| <70% | 0-7分 | 大量遗漏 |

### 必需信息（⭐高优先级）
- [ ] 主诉/就诊原因
- [ ] 病程时间
- [ ] 当前用药（药名+剂量+频次）
- [ ] 近期检查结果
- [ ] 重要症状（如体重下降）
- [ ] 既往治疗史
- [ ] 家族史

---

## 3️⃣ 临床实用性（20分）

**评分原则：检查是否突出临床关键点**

### 评分参考（每突出1项 +2-3分）

**必须突出的关键点：**
- [ ] 血糖控制情况（当前值+目标值）
- [ ] 体重异常变化（快速下降需警示）
- [ ] 当前完整用药方案（集中列出）
- [ ] 并发症线索（如泡沫尿→肾病）
- [ ] 强家族史（影响预后）
- [ ] 有"⚠️重点关注"区域

### 分数等级

| 分数 | 等级 | 特征 |
|------|------|------|
| 18-20分 | 优秀 | 重点突出，有风险提示，助决策 |
| 15-17分 | 良好 | 重要信息基本突出 |
| 11-14分 | 中等 | 信息平铺，重点不突出 |
| <11分 | 较差 | 缺少临床思维 |

---

## 4️⃣ 结构清晰度（15分）

**评分原则：评估"好不好用"**

### 优秀结构特征（每项 +2-3分）
- [ ] 有清晰章节（基本信息、主诉、现病史等）
- [ ] 使用标题层级（#, ##, ###）
- [ ] 相关信息归类（所有用药在一起）
- [ ] 时间线清晰（起病→治疗→变化）
- [ ] 使用列表/表格
- [ ] 能快速定位信息

### 分数等级

| 分数 | 说明 |
|------|------|
| 14-15分 | 层次分明，一目了然 |
| 11-13分 | 基本清晰，略有混乱 |
| 8-10分 | 有分节但不清晰 |
| <8分 | 结构混乱 |

---

## 5️⃣ 语言专业性（10分）

**评分原则：从10分开始，发现问题即扣分**

### 常见问题扣分

| 问题类型 | 扣分 | ❌ 错误示例 | ✅ 正确示例 |
|---------|------|----------|----------|
| 冗余表达 | -0.5 | "患者的糖尿病疾病" | "糖尿病" |
| 口语化 | -1 | "有点高" | "升高" |
| 不专业 | -1 | "血糖不好" | "血糖控制不佳" |

### 检查清单
- [ ] 医学术语准确
- [ ] 表达简洁无冗余
- [ ] 语句通顺
- [ ] 避免口语化
- [ ] 专业性强

---

## 🎯 快速评分流程（8-10分钟/案例）

### Step 1：准备（1分钟）
- 阅读原始问诊记录
- 标记10-15个关键信息点

### Step 2：评分（6分钟）

**① 准确性（2分钟）**
- 从30分开始
- 逐项核对，发现错误即扣分

**② 完整性（1分钟）**
- 统计覆盖率
- 对照表格打分

**③ 实用性（1分钟）**
- 检查是否突出6个关键点
- 每突出1项 +2-3分

**④ 结构（1分钟）**
- 整体评估可用性
- 对照特征打分

**⑤ 语言（1分钟）**
- 从10分开始
- 发现问题即扣分

### Step 3：总结（2分钟）
- 计算总分
- 记录主要问题
- 判断临床可用性

---

## 📈 分数等级

| 总分 | 等级 | 临床可用性 |
|------|------|-----------|
| **90-100** | 🏆 A+ 优秀 | ✅ 可直接使用 |
| **80-89** | ⭐ A 良好 | ✅ 稍作修改 |
| **70-79** | 👍 B 中等偏上 | ⚠️ 需要修改 |
| **60-69** | 😐 C 中等 | ⚠️ 需较多修改 |
| **50-59** | 😕 D 及格 | ❌ 需大量修改 |
| **<50** | ❌ F 不及格 | ❌ 基本不可用 |

## 💡 评分注意事项

### ✅ 应该做的
1. 严格对照标准，逐项核对
2. 同样错误给予同样扣分
3. 记录扣分/加分依据
4. 站在临床医生角度评估

### ❌ 不要做的
1. 凭感觉打分
2. 过于宽松或严格（偏离标准）
3. 受格式影响（内容准确即可）
4. 疲劳时降低标准
